NOTES

    How to use this

  • In the CodePen version, you have to add your code in backticks to an array called input. That is only for anyone who wants to test this out before cloning my repo.
  • Once you do that, copy the code under Code to copy for dark code block.
  • Paste that code into your HTML file below the first <pre> block in the comment field so the <li> tags break to a new line.
  • Then remove the opening and closing <li> tags and paste the remaining code into your <pre> block - DONE! The code in the <pre> tag will look funky but you should not have to touch it after pasting it except for using TAB and/or SPACEBAR to align things.
