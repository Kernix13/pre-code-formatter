NOTES

    How to use this

  • At this point, you have to add your code to script.js in backticks to an array called input or codeToConvert. I do not like this approach.
  • Then save script.js and copy the code under Code to copy for dark code block.
  • Paste that code anywhere into your HTML file so the li tags break to a new line. Then remove the opening and closing li tags and paste the remaining code into your pre block - DONE! The code in the pre tag will look funky but you should not have to touch it after pasting it.
