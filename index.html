NOTES

    How to use this

  • In the CodePen version, you have to add your code in backticks to an array called input. That is only for testing.
  • Once you do that, copy the code under Code to copy for dark code block.
  • Paste that code anywhere into your HTML file so the li tags break to a new line. Then remove the opening and closing li tags and paste the remaining code into your pre block - DONE! The code in the pre tag will look funky but you should not have to touch it after pasting it except for using TAB and/or SPACEBAR to align things.
