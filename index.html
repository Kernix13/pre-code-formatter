HTML Pre Tag Formatter

Output span tags for use in a pre tag for code blocks in a blog post. The first code block has formatting for a dark background with classes for different languages. I haven't figured out how to add a line break for each span tag:

Example with classes for formatting:

          const users = [ 
           { name: "John_&_Jane", age: 34 }, 
           { name: "Amy_<>_1", age: 20 }, 
           { name: "SINGLE-QUOTE_'_1", age: 10 } 
          ];

Example without any classes for formatting:

          const users = [ 
           { name: "John", age: 34 }, 
           { name: "Amy", age: 20 }, 
           { name: "camperCat", age: 10 } 
          ];

This is what code blocks will look like for anything that is not PHP, CSS, HTML or Javascript such as .htaccess file commands, markdown, etc.

        const users = [
         { name: "John_&_Jane", age: 34 },
         { name: "Amy_<>_1", age: 20 },
         { name: "SINGLE-QUOTE_'_1", age: 10 }
        ];
