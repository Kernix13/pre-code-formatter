HTML Pre Tag Formatter

Output span tags for use in a pre tag for code blocks in a blog post. The first code block has formatting for a dark background with classes for different languages.

Code to copy for dark code block

Dark block example:

        <!-- Manual -->
        <p id="test" class="something" required>words here</p>
        <title>Code Formatter</title>
        <img class="test-img" src="./img/file.jpeg">

        <!-- RegEx: htmlTag -->
        <p id="something" class="test" required>words here</p>
        <title>Code Formatter</title>
        <img class="test-img" src="./img/file.jpeg">
        
        <!-- RegEx: htmlAttr -->
        <p id="something" class="test" required>words here</p>
        <title>Code Formatter</title>
        <img class="test-img" src="./img/file.jpeg">

        <!-- RegEx: htmlBoolAttr -->
        <p id="something" class="test" required>words here</p>

        <!-- RegEx: dblQuote -->
